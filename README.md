## üöÄ About The Project

Welcome to my personal portfolio website! This project is designed to be a clean, responsive, and interactive showcase of my skills, projects, and professional journey. Whether you are a recruiter, a fellow developer, or just curious, I hope you enjoy exploring my work.

### ‚ú® Key Features

*   **Responsive Design:** Looks great on desktops, tablets, and mobile devices.
*   **Dynamic Animations:** Smooth transitions and particle effects using `react-tsparticles`.
*   **Modern UI:** Built with `React-Bootstrap` for a sleek and professional look.
*   **PDF Resume Viewer:** Integrated `react-pdf` to view and download my CV directly.
*   **GitHub Integration:** Shows my latest GitHub contributions and stats.

## üíª Getting Started

Want to run this portfolio on your local machine or use it as a template for your own? Follow these simple steps!

### Prerequisites

Make sure you have **Node.js** installed on your system. You can download it [here](https://nodejs.org/).

### Installation & Run

1.  **Clone the repository**
    ```bash
    git clone https://github.com/abhhiixxhek/Abhishek-s-Portfolio.git
    ```

2.  **Navigate to the project directory**
    ```bash
    cd Abhishek-s-Portfolio
    ```

3.  **Install dependencies**
    ```bash
    npm install
    ```

4.  **Start the application**
    ```bash
    npm start
    ```
    The app will open automatically at `http://localhost:3000`.

## üé® how to Customize

If you're forking this repo to use for your own portfolio, here are the key files you need to update:

1.  **Personal Info Update:**
    *   `src/components/Home/Home2.js`: Intro text and social links.
    *   `src/components/About/AboutCard.js`: Your professional bio.
    *   `src/components/Footer.js`: Copyright name and social links.
    *   `src/components/Navbar.js`: Navigation links if you add/remove pages.

2.  **Resume:**
    *   Replace `src/Assets/AbhishekKumarMAIN.pdf` with your own resume PDF.
    *   *Note:* Ensure the filename matches or update the import in `src/components/Resume/ResumeNew.js`.

3.  **Images:**
    *   Update `src/Assets/avatar.svg`, `src/Assets/home-main.svg`, and other images in the `src/Assets` folder with your own.
